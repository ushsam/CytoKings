# XGBoost Sex Classification Pipeline

## Project Overview
This pipeline uses **XGBoost** to predict biological sex (Male/Female) from **18 cytokine expression measurements** using standardized immune biomarkers from the Grifols clinical dataset.

## Target Variable
- **SEXC**: Biological Sex (Binary Classification)
  - 1 = Male
  - 0 = Female

## Features
18 immune signaling molecules (cytokines):
- **Pro-inflammatory**: TNF-alpha, IL-1beta, IL-6, IL-12p70, IFN-gamma
- **Anti-inflammatory**: IL-10, IL-4, IL-5
- **Regulatory**: IL-17A, IL-21, IL-22, IL-23, IL-27
- **Other**: GM-CSF, IL-2, IL-9, IL-13, IL-18

## Pipeline Components

### 1. Data Loading
- Loads preprocessed standardized cytokine features from `X_processed.npy`
- Loads binary sex labels from `y_labels.npy`
- Both files generated by the PCA+KNN pipeline

### 2. Hyperparameter Tuning
- Grid search over:
  - `n_estimators`: [100, 200, 300]
  - `max_depth`: [3, 5, 7, 9]
  - `learning_rate`: [0.01, 0.05, 0.1]
  - `subsample`: [0.8, 0.9]
  - `colsample_bytree`: [0.8, 0.9]
- Uses 5-fold cross-validation for optimization

### 3. Model Evaluation
- Cross-validation accuracy
- Train accuracy, precision, recall, F1-score
- ROC-AUC score
- Confusion matrix
- Classification report (sensitivity, specificity)

### 4. Feature Importance
- Identifies which cytokines are most predictive of sex
- Biological interpretation of immune signals

### 5. Visualization
- Feature importance (top 10 cytokines)
- Confusion matrix heatmap
- ROC curve
- Cross-validation fold scores

## Output Files
- `xgboost_sex_classifier.json` - Trained model (can be loaded later)
- `feature_importance.csv` - Ranked cytokines by importance
- `xgboost_results_summary.csv` - Key metrics and best parameters
- `xgboost_results_visualization.png` - 4-panel results figure
